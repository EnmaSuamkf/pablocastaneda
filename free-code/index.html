Runs on Linux and macOS
The installer sets up the CLI (on macOS, also the desktop app). The VS Code / Cursor plugin is a quick separate step you do by hand afterwards. Both installers live in the installation/ folder.
Funciona en Linux y macOS
El instalador configura la CLI (en macOS, también la app de escritorio). El plugin de VS Code / Cursor es un paso separado y rápido que haces manualmente después. Ambos instaladores están en la carpeta installation/.

1. Fast installation1. Instalación rápida

Run the script for your platform from the installation/ folder:Ejecuta el script para tu plataforma desde la carpeta installation/:

# Linux — installs the CLI
bash ./installation/install-free-code-linux.sh

# macOS — installs the CLI + desktop app
bash ./installation/install-free-code-mac.command

On macOS you can also double-click install-free-code-mac.command in Finder.En macOS también puedes hacer doble clic en install-free-code-mac.command desde Finder.

Usa proveedores de LLM en la nube cuando quieras, o ejecuta en entornos gestionados — con capas de seguridad como Damage-Control y autorización de MCP donde tu política lo requiera.
